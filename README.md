# NOVA ATELIER 🌌
**An Industrial-Sovereign Ecommerce Experience**

Welcome to **Nova Atelier**, a high-end digital storefront designed with a "Black Label" aesthetic. This project serves as a comprehensive demonstration of professional "seller readiness" for freelance web development, focusing on advanced UI/UX interactions and robust JavaScript logic[cite: 1].

---

## 🛠 Features

*   **Industrial Design System:** A sophisticated "Sovereign" visual language featuring high-contrast themes, editorial layouts, and architectural strategy[cite: 1].
*   **Object-Oriented Logic:** Built using a custom `NovaStore` Class to manage product rendering, cart state, and wishlist functionality[cite: 3].
*   **Persistent Shopping Cart:** Fully functional ecommerce checkout flow with data persistence via `LocalStorage`[cite: 2, 3].
*   **Premium Interactions:**
    *   Dynamic page loaders and custom cursor themes that adapt to light/dark sections[cite: 2, 3].
    *   Scroll-triggered animations using `IntersectionObserver`[cite: 3].
    *   Debounced product search and category filtering[cite: 3].
*   **Responsive Quad-Matrix Grid:** A custom CSS grid system designed to provide a seamless experience across desktop and mobile devices[cite: 2].
